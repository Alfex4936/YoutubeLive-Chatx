# yt-chatx

Youtube Live 채팅을 API 없이 실시간 읽기

진행 중...

## 개발 스택
- Redis - Realtime statistics
- Playwright - Chrome mock
- Spring boot 3.4 (VT on) - API server
- Java 23
- Postgres 17 - Profanity, logs etc
- Websocket (SockJS) - Realtime get

## 기능
- [x] Youtube Live 채팅 읽기
- [x] 실시간 통계
  - [x] 채팅 수 (x 초당)
  - [x] 욕설 수 (x 초당)
  - [x] 키워드 랭킹
